# SpringCore
Core library for all of Springpoint Network's plugins.

## Features
0. Annotation-based command library
0. JSON serialization, with JSON configuration and localization.
0. Module API
0. Inventory GUIs
0. Particle API
0. Scoreboard API
0. Title and subtitle API
0. Integration API

* AmpMenus by ampayne2
